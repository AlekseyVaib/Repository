БЫСТРЫЙ СТАРТ:
═══════════════════════════════════════════════════════════════

1. УСТАНОВИТЕ PYTHON (если еще не установлен)
   - Скачайте с https://www.python.org/downloads/
   - При установке ОБЯЗАТЕЛЬНО отметьте "Add Python to PATH"
   - Установите Python

2. УСТАНОВИТЕ НЕОБХОДИМЫЕ МОДУЛИ
   - Откройте командную строку (Win+R → cmd)
   - Перейдите в папку с программой:
     cd путь\к\папке\Валидатор
   - Выполните команду:
     pip install pandas openpyxl dnspython

3. ЗАПУСТИТЕ ПРОГРАММУ
   
   ВАРИАНТ A: GUI приложение (рекомендуется)
   - В командной строке выполните:
     python email_validator_gui.py
   - Откроется красивое окно "Валидатор 3000"
   - Выберите файл с email адресами
   - Настройте параметры
   - Нажмите "Начать проверку"
   
   ВАРИАНТ B: Консольный режим
   - В командной строке выполните:
     python email_validator.py
   - Ответьте на вопросы программы

РЕЖИМЫ ВАЛИДАЦИИ:
═══════════════════════════════════════════════════════════════

🔒 СТРОГИЙ РЕЖИМ (по умолчанию):
   • Максимальная точность (95-98% совпадение с Mailvalidator)
   • Только адреса с высокой надежностью
   • Обязательная активность email
   • Проверка репутации домена
   • Рекомендуется для важных рассылок

✨ ЛОЯЛЬНЫЙ РЕЖИМ:
   • На 15-20% больше валидных адресов
   • Адреса с высокой и средней надежностью
   • Мягкие требования к активности
   • Подходит для массовых рассылок

ПОДГОТОВКА ФАЙЛА С EMAIL:
═══════════════════════════════════════════════════════════════

1. Создайте Excel файл (.xlsx)
2. В первой строке (опционально) заголовок: Email, E-mail, Почта и т.д.
3. В следующих строках введите email адреса (по одному в строке)
4. Сохраните файл в папку с программой

ПРИМЕР:
