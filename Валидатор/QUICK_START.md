# Быстрый старт

## Запуск через терминал (командную строку)

### 1. Откройте командную строку

**Windows:**
- Нажмите `Win + R`
- Введите `cmd` и нажмите Enter
- Или найдите "Командная строка" в меню Пуск

### 2. Перейдите в папку с программой

```bash
cd C:\Users\ВашеИмя\Desktop\Валидатор
```

(Замените `ВашеИмя` на ваше имя пользователя)

### 3. Запустите программу

#### Вариант A: GUI приложение (с графическим интерфейсом)

```bash
python email_validator_gui.py
```

Откроется окно с удобным интерфейсом для работы.

#### Вариант B: Консольное приложение (интерактивный режим)

```bash
python email_validator.py
```

Программа задаст вопросы:
1. Название файла (по умолчанию: `emails`)
2. Режим валидации (1 - строгий, 2 - лояльный)
3. Выполнять SMTP проверку? (да/нет)
4. Считать валидными catch-all почты? (да/нет)
5. Сколько почт проверить? (оставьте пустым для всех)

#### Вариант C: С параметрами командной строки

```bash
# Базовый запуск с указанием файла
python email_validator.py --input emails.xlsx

# С выходным файлом
python email_validator.py --input emails.xlsx --output results.xlsx

# Без SMTP проверки (быстрее)
python email_validator.py --input emails.xlsx --no-smtp

# С таймаутом
python email_validator.py --input emails.xlsx --timeout 15
```

## Режимы валидации

### Строгий режим (по умолчанию)
- Максимальная точность
- Только адреса с высокой надежностью
- Проверка репутации домена
- Совпадение с Mailvalidator ~95-98%

### Лояльный режим
- На 15-20% больше валидных адресов
- Адреса с высокой и средней надежностью
- Мягкие требования к активности
- Подходит для массовых рассылок

## Результат

После проверки будет создан Excel файл с результатами:
- Название: `название_входного_файла_YYYYMMDD_HHMMSS.xlsx`
- Содержит все проверенные адреса с детальной информацией
- Цветовое форматирование для удобства

## Помощь

Если возникли проблемы:
1. Убедитесь, что Python установлен: `python --version`
2. Убедитесь, что установлены модули: `pip install pandas openpyxl dnspython`
3. Проверьте, что файл с email адресами находится в папке с программой
4. Смотрите подробную инструкцию в файле `GUIDE.md`
