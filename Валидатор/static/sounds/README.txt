Звуки для веб-валидатора V3.1
================================

Положите в эту папку свои аудиофайлы — они будут проигрываться при действиях на сайте.

Имена файлов (поддерживаются форматы .mp3, .wav, .ogg):

  upload.mp3   — когда выбран или перетащен файл (зона загрузки)
  start.mp3    — когда нажата кнопка «Начать проверку» и валидация стартовала
  complete.mp3 — когда проверка успешно завершена

Если файла с нужным именем нет, будет использован встроенный короткий звуковой сигнал.

Рекомендации:
- Короткие файлы (0.5–2 сек) лучше не отвлекают.
- Формат MP3 подходит для всех браузеров.
